自宅ラボ構築

Proxmox / Tailscale / Cloudflare Tunnel

Proxmox VETailscaleCloudflare TunnelDockerNginxgit hook

overview

中古のPCに Proxmox VE を導入し、自分専用のインフラ基盤を構築しました。VMやLXCコンテナを用途ごとに分け、ポートフォリオの公開や検証環境として運用しています。

構成

  • Proxmox VE をハイパーバイザとして導入し、VM / LXCを役割ごとに分離
  • Tailscale を導入し、外出先からも自宅環境へ安全にSSH・管理アクセス
  • Cloudflare Tunnel でポートフォリオサイトを公開し、グローバルへのポート開放を行わない構成に
  • Docker + Nginx でアプリケーションを実行し、用途ごとにコンテナを分離

自動デプロイの仕組み

GitHubリポジトリへの git push をきっかけに、サーバ側の git hook が git pull → docker build → コンテナ再起動までを自動で行うようにしています。手元での変更がそのまま公開環境に反映されるため、検証から公開までのサイクルを短くできています。

developer │ git push ▼ ┌──────────────┐ │ GitHub repo │ └──────┬───────┘ │ pull (git hook) ┌──────┴───────┐ │ Proxmox VM │ │ docker build│ │ docker run │ └──────────────┘ │ ▼ Cloudflare Tunnel 経由で公開

今後の予定

  • 監視・バックアップ基盤の強化
  • 検証用ネットワーク機器(L3スイッチ等)の自宅ラボへの組み込み
  • 学内ネットワーク運用で得た知見をホームラボの設計にも反映
